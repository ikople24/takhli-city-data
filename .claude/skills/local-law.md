## อำนาจหน้าที่ของเทศบาลเมือง (มาตรา 50 พ.ร.บ.เทศบาล)

**หน้าที่บังคับ (ต้องทำ):**
1. รักษาความสะอาด/ความเป็นระเบียบเรียบร้อย
2. ให้มีน้ำสะอาดหรือน้ำประปา
3. ให้มีโรงฆ่าสัตว์
4. ให้มีการบำรุงรักษาทางบกและทางน้ำ
5. ให้มีเครื่องดับเพลิงและการระงับอัคคีภัย
6. ให้มีการรักษาความสงบเรียบร้อยของประชาชน

**หน้าที่เลือกทำ (ทำได้):**
- โรงพยาบาล สถานพยาบาล ห้องสมุด พิพิธภัณฑ์
- การวางผังเมือง สวนสาธารณะ สนามกีฬา
- การพาณิชย์ สาธารณูปการ ส่งเสริมอาชีพ

## โครงสร้างเทศบัญญัติ
```
เทศบัญญัติเทศบาลเมืองตาคลี
เรื่อง [ชื่อ] พ.ศ. [ปี]
-----------------------------
อาศัยอำนาจตามมาตรา [X] แห่งพระราชบัญญัติเทศบาล
พ.ศ. ๒๔๙๖ และมาตรา [X] แห่ง[กฎหมายอื่น]...
สภาเทศบาลเมืองตาคลีได้มีมติในการประชุมสมัย[สามัญ/วิสามัญ]
ครั้งที่ [X] ประจำปี พ.ศ. [ปี] เมื่อวันที่ [วันที่]...

หมวด ๑ บทนิยาม
มาตรา ๑ เทศบัญญัตินี้เรียกว่า "..."
มาตรา ๒ ให้ใช้บังคับตั้งแต่วันถัดจากวันประกาศ...
มาตรา ๓ ในเทศบัญญัตินี้ "[คำ]" หมายความว่า...

หมวด ๒ [เนื้อหา]
...

หมวด [สุดท้าย] บทกำหนดโทษ
มาตรา [X] ผู้ใดฝ่าฝืน... ต้องระวางโทษ...

ประกาศ ณ วันที่...
(นายกเทศมนตรีเมืองตาคลี)
```

## ระยะเวลาตอบสนองตาม พ.ร.บ.อำนวยความสะดวกฯ
- คำขออนุญาตทั่วไป: ≤ 30 วันทำการ
- ต้องแจ้งผลภายในกำหนดหรือขอขยายเวลาพร้อมเหตุผล
- ต้องมีคู่มือประชาชนเผยแพร่ทุกบริการ
