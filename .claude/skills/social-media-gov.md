# ทักษะ: สื่อโซเชียลสำหรับราชการ

## หลักการ CLEAR สำหรับราชการ

```
C — Clear      ชัดเจน ไม่กำกวม อ่านรู้เรื่องทันที
L — Local      เชื่อมกับชีวิตประชาชนในพื้นที่ตาคลี
E — Engaging   น่าสนใจ มีคุณค่า ไม่ใช่แค่ประกาศ
A — Actionable บอกสิ่งที่ประชาชนทำได้ชัดเจน
R — Reliable   ข้อมูลถูกต้อง มีแหล่งอ้างอิง
```

## โครงสร้างโพสต์แต่ละประเภท

### ประกาศข่าวบริการ
```
📢 [หัวเรื่อง ดึงดูด ≤ 10 คำ]

[เนื้อหาหลัก 2-3 ประโยค ตอบ ใคร ทำอะไร ที่ไหน เมื่อไร]

✅ สิ่งที่ประชาชนต้องทำ/รู้:
• [ข้อ 1]
• [ข้อ 2]

📞 สอบถาม: [กอง] เทศบาลเมืองตาคลี โทร. 0-5621-9299
🕐 เปิดทำการ: จันทร์-ศุกร์ 08:30-16:30 น.

#เทศบาลตาคลี #[หัวข้อ] #นครสวรรค์
```

### รณรงค์/ให้ความรู้
```
💡 รู้หรือไม่? [ข้อเท็จจริงน่าสนใจ]

[อธิบาย 2-3 ประโยค]

🔍 วิธีป้องกัน/ดูแล:
1️⃣ [ข้อ 1]
2️⃣ [ข้อ 2]
3️⃣ [ข้อ 3]

[Call to Action — แชร์ให้เพื่อนบ้าน/แจ้งเทศบาล]

#[หัวข้อ] #เทศบาลตาคลี
```

### ประชาสัมพันธ์กิจกรรม/งาน
```
🎉 [ชื่องาน]

📅 วันที่: [วัน เดือน ปี]
⏰ เวลา: [HH:MM] น.
📍 สถานที่: [ชื่อสถานที่ ระบุชัดเจน]
👥 สำหรับ: [กลุ่มเป้าหมาย]
💰 ค่าใช้จ่าย: ฟรี / [จำนวน] บาท

[ไฮไลต์กิจกรรม 2-3 รายการ]

📲 สมัคร/สอบถาม: [ช่องทาง]

#[ชื่องาน] #เทศบาลตาคลี
```

### แจ้งผลการดำเนินงาน
```
✅ อัปเดตงาน: [ชื่อโครงการ/งาน]

[สรุปผลที่ทำได้ 2-3 ประโยค]
[ผลที่ประชาชนได้รับประโยชน์]

📸 ดูภาพเพิ่มเติม: [ลิงก์ถ้ามี]

เทศบาลเมืองตาคลี พัฒนาเพื่อประชาชน 💙
#เทศบาลตาคลี #[หัวข้อ]
```

```
📋 ข้อมูลบริการต่างๆ
```

```
เรื่องเร่งด่วน: โทร. 0-5621-9299
ฉุกเฉิน/ดับเพลิง: โทร. 199
เทศบาลเมืองตาคลี
```

## หลักการ DO & DON'T

| ✅ DO | ❌ DON'T |
|------|---------|
| เขียนในนามองค์กร ไม่ใช่บุคคล | แสดงความคิดเห็นส่วนตัว |
| ตอบกลับคอมเมนต์/ข้อความภายใน 1 วันทำการ | ปล่อยคอมเมนต์ค้างนาน |
| ใช้รูปภาพคุณภาพดีและได้รับอนุญาต | ใช้รูปติด watermark บริษัทอื่น |
| ตรวจสอบข้อมูลก่อนโพสต์ | แชร์ข่าวที่ยังไม่ได้ยืนยัน |
| emoji เสริมบุคลิก ≤ 3 ตัว/โพสต์ | emoji มากเกินจนดูไม่เป็นทางการ |
| แท็กหน่วยงานที่เกี่ยวข้อง | แท็กบุคคลโดยไม่ได้รับอนุญาต |
