# ทักษะ: จัดซื้อจัดจ้างภาครัฐ

## วิธีการจัดซื้อจัดจ้าง (เลือกตามวงเงิน)

```
วงเงิน ≤ 5,000 บาท
└── ซื้อ/จ้างได้เลย (บันทึกการจ่าย)

วงเงิน 5,001 – 100,000 บาท
└── เฉพาะเจาะจง (สืบราคา ≥ 1 ราย)

วงเงิน 100,001 – 500,000 บาท
└── เฉพาะเจาะจง (สืบราคา ≥ 3 ราย) หรือ e-shopping

วงเงิน 500,001 บาท ขึ้นไป
└── ประกวดราคาอิเล็กทรอนิกส์ (e-bidding)
    หรือคัดเลือก/พิเศษ (กรณีมีเหตุพิเศษ)
```

## โครงสร้าง TOR (Terms of Reference)

```markdown
ขอบเขตของงาน (Terms of Reference)
โครงการ: [ชื่อโครงการ]
หน่วยงานเจ้าของโครงการ: [กอง] เทศบาลเมืองตาคลี

1. วัตถุประสงค์
   [ระบุเป้าหมายที่วัดได้]

2. ขอบเขตงาน (Scope of Work)
   [รายละเอียดงานที่ต้องทำ ระบุให้ชัดเจน]

3. คุณลักษณะเฉพาะ (Specification)
   [มาตรฐาน วัสดุ ขนาด คุณภาพ อ้างอิง มอก./DIN/ASTM]

4. ระยะเวลาดำเนินการ
   [จำนวนวัน] วันนับจากวันลงนามในสัญญา

5. สถานที่ดำเนินการ
   [ระบุที่อยู่/พิกัด]

6. การส่งมอบงาน
   [งวดที่ 1/2/3 พร้อมรายละเอียดงานแต่ละงวด]

7. การรับประกันผลงาน
   รับประกันไม่น้อยกว่า [X] ปี นับจากวันที่ตรวจรับงาน

8. คุณสมบัติผู้เสนอราคา
   [ใบอนุญาต ประสบการณ์ หลักฐานที่ต้องใช้]

9. เกณฑ์การประเมิน
   ราคาต่ำสุด / คุณภาพและราคา (ระบุสัดส่วน)
```

## รายการตรวจสอบก่อนประกาศจัดซื้อ
- [ ] ตรวจสอบงบประมาณในแผนการจัดซื้อจัดจ้างประจำปี
- [ ] จัดทำราคากลาง (ท้องตลาด/หน่วยงานราชการ/ราคามาตรฐานครุภัณฑ์)
- [ ] TOR ผ่านความเห็นชอบจากหัวหน้าหน่วยงาน
- [ ] เผยแพร่แผนบน www.gprocurement.go.th (วงเงิน > 5 แสน: ล่วงหน้า ≥ 3 วันทำการ)
- [ ] ตั้งคณะกรรมการกำหนดราคากลาง (วงเงิน > 1 ล้าน)

## แบบรายงานการตรวจรับงาน
```
บันทึกการตรวจรับพัสดุ/งานจ้าง
วันที่: [วัน/เดือน/ปี]
สัญญาเลขที่: [เลขที่] ลงวันที่ [วันที่]
ผู้ขาย/ผู้รับจ้าง: [ชื่อ]
รายการพัสดุ/งาน: [รายละเอียด]
จำนวน/ปริมาณ: [ระบุ] ราคารวม: [บาท]
ผลการตรวจรับ: ✅ ถูกต้องครบถ้วน / ❌ บกพร่อง ([ระบุ])

คณะกรรมการตรวจรับ:
1. [ชื่อ] [ตำแหน่ง]  ลายมือชื่อ ___________
2. [ชื่อ] [ตำแหน่ง]  ลายมือชื่อ ___________
```

## กฎสำคัญที่ต้องจำ
- **ห้าม** แบ่งซื้อแบ่งจ้างเพื่อหลีกเลี่ยงวิธีการจัดซื้อจัดจ้าง
- **ห้าม** กำหนด spec ที่เอื้อผู้ขายรายใดรายหนึ่ง
- **ต้อง** เผยแพร่ผลการจัดซื้อจัดจ้างบน EGP ภายใน 30 วัน
- **ต้อง** เก็บเอกสารไว้ ≥ 5 ปี หลังสิ้นสุดสัญญา
