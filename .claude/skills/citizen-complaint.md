# ทักษะ: จัดการข้อร้องเรียนประชาชน

## ประเภทคำร้องและผู้รับผิดชอบ

| ประเภทเรื่องร้องเรียน | กองที่รับผิดชอบ | ระยะเวลา |
|--------------------|---------------|---------|
| ถนน/ทางเท้า/ไฟส่องสว่าง | กองช่าง | 15 วันทำการ |
| น้ำท่วม/ท่อระบายน้ำ | กองช่าง | 7 วันทำการ |
| ขยะ/กลิ่นเหม็น | กองสาธารณสุข | 3 วันทำการ |
| สุนัขจรจัด/สัตว์รบกวน | กองสาธารณสุข | 7 วันทำการ |
| เสียงรบกวน/มลภาวะ | กองสาธารณสุข | 7 วันทำการ |
| ภาษี/ค่าธรรมเนียม | กองคลัง | 15 วันทำการ |
| ก่อสร้างผิดกฎหมาย | กองช่าง | 15 วันทำการ |
| สวัสดิการ/เบี้ยยังชีพ | กองสวัสดิการ | 15 วันทำการ |
| ร้องเรียนเจ้าหน้าที่ | สำนักปลัด | 30 วันทำการ |

## ขั้นตอนการจัดการคำร้อง

```
รับคำร้อง (ด้วยตนเอง/โทร/เว็บ/Facebook)
    ↓
ลงทะเบียนรับเรื่อง (วันที่รับ เลขที่ ชื่อผู้ร้อง เรื่อง)
    ↓
จำแนกประเภท + มอบหมายกองที่รับผิดชอบ
    ↓
กองตรวจสอบ/ดำเนินการ
    ↓
แจ้งผลให้ผู้ร้องทราบ (ภายในระยะเวลากำหนด)
    ↓
ปิดเรื่อง (บันทึกผล) / ส่งต่อ (กรณีเกินอำนาจ)
```

## แบบฟอร์มรับเรื่องร้องเรียน
```
เลขที่รับ: [ปี-ลำดับ]          วันที่รับ: [วัน/เดือน/ปี]
ช่องทาง: □ ด้วยตนเอง □ โทรศัพท์ □ เว็บไซต์ □ Facebook □ อื่นๆ

ผู้ร้องเรียน:
  ชื่อ-สกุล: [ชื่อ]
  ที่อยู่/ชุมชน: [ชุมชน/บ้านเลขที่]
  โทรศัพท์: [เบอร์] (ไม่บังคับ)

เรื่องที่ร้องเรียน: [ประเภท]
รายละเอียด: [สรุปปัญหา]
สถานที่เกิดเหตุ: [ระบุ]

มอบหมาย: กอง[ชื่อกอง]  กำหนดแล้วเสร็จ: [วันที่]
ผู้รับมอบหมาย: [ชื่อ]

ผลการดำเนินการ: [บันทึกผล]
วันที่แล้วเสร็จ: [วันที่]  สถานะ: □ แก้ไขแล้ว □ ส่งต่อ □ อยู่ระหว่างดำเนินการ
```

## รูปแบบการตอบกลับผู้ร้อง

### ตอบรับเรื่อง (SMS/Line ส่งทันที)
```
เทศบาลเมืองตาคลี รับเรื่องร้องเรียนของท่านแล้ว
เลขที่ [เลขที่] เรื่อง[ชื่อเรื่อง]
กำหนดแจ้งผลภายใน [X] วันทำการ
สอบถาม: 0-5621-9299
```

### แจ้งผลการดำเนินการ (หนังสือ/โทรศัพท์)
```
เรียน คุณ[ชื่อ]

ตามที่ท่านได้ร้องเรียน เรื่อง[เรื่อง] เมื่อวันที่[วันที่]
เทศบาลเมืองตาคลีได้ดำเนินการ[รายละเอียด] เรียบร้อยแล้ว
เมื่อวันที่ [วันที่]

หากมีข้อสงสัยติดต่อ [ชื่อเจ้าหน้าที่] โทร. [เบอร์]
```

## สถิติที่ควรรายงานประจำเดือน
- จำนวนคำร้องที่รับ / แก้ไขแล้ว / ค้างอยู่
- เรื่องที่พบมากที่สุด 5 อันดับ
- อัตราการแก้ไขตามกำหนดเวลา (%)
- ความพึงพอใจของผู้ร้อง (ถ้าสำรวจ)

## กรณีเกินอำนาจเทศบาล
| เรื่อง | ส่งต่อไปที่ |
|-------|------------|
| ถนนหลวง/ทางหลวง | แขวงทางหลวงนครสวรรค์ที่ 2 |
| น้ำประปาส่วนภูมิภาค | การประปาส่วนภูมิภาค สาขาตาคลี |
| ไฟฟ้า | การไฟฟ้าส่วนภูมิภาค อ.ตาคลี |
| ความมั่นคง/อาชญากรรม | สถานีตำรวจภูธรตาคลี |
| ที่ดิน/ทะเบียน | สำนักงานที่ดินจังหวัดนครสวรรค์ สาขาตาคลี |
