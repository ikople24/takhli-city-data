# ทักษะ: GIS และข้อมูลเชิงพื้นที่

## ชั้นข้อมูล (Layers) สำคัญสำหรับ ทม.ตาคลี

### ชั้นข้อมูลพื้นฐาน (Base Layers)
```
□ ขอบเขตเทศบาล / 23 ชุมชน
□ ถนน ทางเท้า ซอย (แบ่งตามสภาพ/มาตรฐาน)
□ อาคาร/สิ่งปลูกสร้าง (ประเภท จำนวนชั้น)
□ แปลงที่ดิน (โฉนด / น.ส.3 / ที่สาธารณะ)
□ แหล่งน้ำ (คลอง บึง แม่น้ำ)
□ ภาพถ่ายทางอากาศ/ดาวเทียม (อัปเดตทุกปี)
```

### ชั้นข้อมูล Smart City
```
□ โครงข่ายท่อประปา (ขนาด อายุ วัสดุ)
□ ระบบระบายน้ำ (ท่อ บ่อพัก ทิศทางไหล)
□ ไฟฟ้าสาธารณะ (จุดติดตั้ง ประเภท สถานะ)
□ กล้อง CCTV (ตำแหน่ง มุมกล้อง รัศมีครอบคลุม)
□ IoT Sensor (ประเภท จุดติดตั้ง สถานะ)
□ จุดเก็บขยะ / เส้นทางรถเก็บขยะ
□ จุดเสี่ยงน้ำท่วม (ระดับความเสี่ยง)
□ สถานที่สำคัญ (โรงเรียน โรงพยาบาล ตลาด วัด)
```

## มาตรฐานข้อมูลที่ใช้

| รายการ | มาตรฐาน |
|--------|--------|
| ระบบพิกัด | WGS84 (EPSG:4326) หรือ UTM Zone 47N (EPSG:32647) |
| รูปแบบไฟล์ | GeoJSON, Shapefile (.shp), GeoPackage (.gpkg) |
| ภาพแรสเตอร์ | GeoTIFF |
| Metadata | ISO 19115 / DCAT-AP |
| API | OGC WMS, WFS, WMTS |

## เครื่องมือ GIS ที่แนะนำสำหรับเทศบาล (ฟรี/ราคาย่อมเยา)

| เครื่องมือ | ใช้ทำ | ต้นทุน |
|-----------|------|--------|
| QGIS | วิเคราะห์ จัดทำแผนที่ | ฟรี |
| Google My Maps | แผนที่ง่ายๆ แชร์ได้ | ฟรี |
| Felt.com | Collaborative mapping | ฟรี (basic) |
| OpenStreetMap | ข้อมูลแผนที่พื้นฐาน | ฟรี |
| Longdo Map API | แผนที่ภาษาไทย | มี free tier |
| Gistda Open Data | ภาพดาวเทียม/ข้อมูลไทย | ฟรี |

## การวิเคราะห์เชิงพื้นที่สำหรับงานเทศบาล

### วิเคราะห์พื้นที่เสี่ยงน้ำท่วม
```
ข้อมูลที่ต้องใช้:
- DEM (Digital Elevation Model) ความละเอียด ≤ 5 เมตร
- ท่อระบายน้ำ (ขนาด / ความจุ)
- จุดน้ำท่วมซ้ำซาก (Historical flood points)
- ปริมาณฝน (สถานีวัดน้ำฝน)

ผลลัพธ์:
- แผนที่ความเสี่ยงน้ำท่วม 3 ระดับ (สูง/กลาง/ต่ำ)
- จุดคอขวดระบบระบาย
- ลำดับความสำคัญในการซ่อมบำรุง
```

### วิเคราะห์เส้นทางรถเก็บขยะ (Route Optimization)
```
Input:  จุดเก็บขยะ 23 ชุมชน + โครงข่ายถนน + เวลาทำการ
Output: เส้นทางที่ระยะทางสั้นที่สุด / ใช้เวลาน้อยที่สุด
เครื่องมือ: QGIS Network Analysis / OR-Tools (Python)
ประโยชน์: ลดค่าน้ำมัน ลด CO₂ เพิ่มรอบการเก็บ
```

### วางแผนติดตั้ง CCTV / Sensor
```
เกณฑ์เลือกตำแหน่ง:
- จุดเสี่ยงอาชญากรรม (Crime hotspot จากสถิติ ตร.)
- จุดตัดถนนสำคัญ / ทางเข้า-ออกเมือง
- รัศมีครอบคลุมไม่ทับซ้อนเกิน 20%
- ระยะห่างจากไฟฟ้าสาธารณะ (ลด cost สายสัญญาณ)
- ไม่ละเมิด PDPA (ไม่ชี้เข้าบ้านเรือนส่วนตัว)
```

## โครงสร้าง TOR ระบบ GIS สำหรับเทศบาล

```
ขอบเขตของงาน: จัดทำระบบ GIS เทศบาลเมืองตาคลี

1. จัดทำชั้นข้อมูลพื้นฐาน [ระบุ layers ที่ต้องการ]
2. พัฒนา Web GIS Portal (เข้าถึงผ่าน browser)
3. อบรมเจ้าหน้าที่ [จำนวน] คน [จำนวน] ชั่วโมง
4. คู่มือการใช้งานและดูแลระบบ (ภาษาไทย)
5. รับประกันระบบ [X] ปี + support [X] ชั่วโมง/ปี

คุณลักษณะขั้นต่ำ:
- รองรับผู้ใช้พร้อมกัน ≥ 20 คน
- ข้อมูลพิกัด WGS84 / UTM Zone 47N
- Export ได้เป็น PDF, PNG, Shapefile, GeoJSON
- Mobile-friendly (เจ้าหน้าที่ใช้ภาคสนามได้)
- เชื่อมต่อ API กับระบบอื่นของเทศบาลได้
```
