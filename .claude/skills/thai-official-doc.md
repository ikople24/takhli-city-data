# ทักษะ: ร่างเอกสารราชการไทย

## ประเภทเอกสารและรูปแบบมาตรฐาน

### 1. หนังสือภายนอก (ติดต่อหน่วยงานอื่น)
```
ที่ นว ๕๒๑๐๑/[เลขที่]        [ชื่อหน่วยงาน]
                               [ที่อยู่]
วันที่ [วัน เดือน ปี พ.ศ. ตัวเลข อักษร]

เรื่อง  [ชื่อเรื่อง กระชับ ไม่เกิน 1 บรรทัด]
เรียน  [ตำแหน่งผู้รับ ไม่ใส่ชื่อบุคคล]
อ้างถึง [ถ้ามี]
สิ่งที่ส่งมาด้วย [ถ้ามี]

    [ย่อหน้าแรก: เหตุที่มีหนังสือ]
    [ย่อหน้ากลาง: รายละเอียด]
    [ย่อหน้าสุดท้าย: สรุป/ขอความร่วมมือ]

จึงเรียนมาเพื่อโปรด[ทราบ/พิจารณา/ดำเนินการ]

                              ขอแสดงความนับถือ

                         ([ชื่อ-สกุลผู้ลงนาม])
                    ([ตำแหน่ง])
[ส่วนงานที่รับผิดชอบ]
โทร. [เบอร์]  โทรสาร [เบอร์]
```

### 2. บันทึกข้อความ (ติดต่อภายในองค์กร)
```
บันทึกข้อความ
ส่วนราชการ  [กอง/ฝ่าย] เทศบาลเมืองตาคลี
ที่  นว ๕๒๑๐๑/[เลขที่]   วันที่  [วัน เดือน ปี]
เรื่อง  [ชื่อเรื่อง]
เรียน  [ตำแหน่ง]

    [เนื้อหา]

([ชื่อ-สกุล])
([ตำแหน่ง])
```

### 3. ประกาศเทศบาล
```
ประกาศเทศบาลเมืองตาคลี
เรื่อง [ชื่อเรื่อง]
--------------------
    [เหตุอ้างอิง อาศัยอำนาจตาม...]

    จึงประกาศให้ทราบโดยทั่วกัน

ประกาศ ณ วันที่ [วัน] เดือน [เดือน] พ.ศ. [ปี]

                    ([ชื่อ-สกุล])
               นายกเทศมนตรีเมืองตาคลี
```

### 4. คำสั่งเทศบาล
```
คำสั่งเทศบาลเมืองตาคลี
ที่ [เลขที่]/[ปี พ.ศ.]
เรื่อง [ชื่อเรื่อง]
--------------------
    อาศัยอำนาจตาม[มาตรา/ข้อ]...
    จึงมีคำสั่ง ดังนี้
    ข้อ ๑ ...
    ข้อ ๒ ...

    ทั้งนี้ ตั้งแต่วันที่ ... เป็นต้นไป

สั่ง ณ วันที่ [วัน เดือน ปี]

                    ([ชื่อ-สกุล])
               นายกเทศมนตรีเมืองตาคลี
```

## กฎการใช้ภาษา

| สิ่งที่ต้องทำ | สิ่งที่ต้องหลีกเลี่ยง |
|-------------|-------------------|
| ตัวเลขไทย ๑ ๒ ๓ ในเอกสารทางการ | ตัวเลขอารบิค 1 2 3 ในหัวข้อหลัก |
| "จึงเรียนมาเพื่อโปรดทราบ" | "จึงแจ้งมาให้ทราบ" |
| ย่อหน้าเว้นวรรค 5 ตัวอักษร | ไม่มีการย่อหน้า |
| ลงวันที่แบบไทย "๖ มิถุนายน ๒๕๖๙" | วันที่แบบสั้น |
| ระบุเบอร์โทร/โทรสารท้ายหนังสือ | ละเลขติดต่อ |

## เลขที่หนังสือ เทศบาลเมืองตาคลี
- **รหัสจังหวัด:** นว (นครสวรรค์)
- **รหัสหน่วยงาน:** ๕๒๑๐๑
- **รูปแบบ:** `นว ๕๒๑๐๑/[ลำดับ]`

## อำนาจลงนาม
| เอกสาร | ผู้ลงนาม |
|--------|---------|
| หนังสือภายนอกระดับสูง | นายกเทศมนตรี |
| หนังสือภายนอกทั่วไป | ปลัดเทศบาล / รองนายก |
| บันทึกข้อความ | ผู้อำนวยการกอง |
| ประกาศ/คำสั่ง | นายกเทศมนตรี |
