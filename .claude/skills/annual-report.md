# ทักษะ: จัดทำรายงานประจำปี

## โครงสร้างรายงานประจำปีเทศบาล

```
รายงานผลการดำเนินงาน
เทศบาลเมืองตาคลี
ประจำปีงบประมาณ พ.ศ. [ปี]
(๑ ตุลาคม [ปีก่อน] – ๓๐ กันยายน [ปีนี้])
```

### บทที่ 1: ข้อมูลทั่วไปของเทศบาล
- ประวัติความเป็นมา / วิสัยทัศน์ / พันธกิจ
- โครงสร้างองค์กรและบุคลากร (จำนวน ตำแหน่ง)
- ข้อมูลพื้นที่ ประชากร ชุมชน

### บทที่ 2: ผลการดำเนินงานตามยุทธศาสตร์
สำหรับแต่ละยุทธศาสตร์ (6 ด้าน) ระบุ:
```
ยุทธศาสตร์ที่ [X]: [ชื่อ]
เป้าหมาย: [ระบุ]
โครงการ/กิจกรรมที่ดำเนินการ:
  1. [ชื่อโครงการ]
     - งบประมาณ: [X] บาท  เบิกจ่าย: [X] บาท ([%])
     - กลุ่มเป้าหมายที่ได้รับประโยชน์: [X] คน/ชุมชน
     - ผลที่ได้: [สรุป]
ผลลัพธ์ภาพรวม: [สรุปผลสำเร็จ/ปัญหาอุปสรรค]
```

### บทที่ 3: รายงานการเงิน
- งบแสดงฐานะการเงิน
- งบรายรับ-รายจ่าย (เทียบงบประมาณที่ตั้ง)
- อัตราการเบิกจ่าย (% ต่อหมวด)
- รายได้แยกประเภท (ภาษี/อุดหนุน/อื่นๆ)

### บทที่ 4: ผลการให้บริการประชาชน
- จำนวนผู้มาขอรับบริการ (แยกตามประเภท)
- ผลสำรวจความพึงพอใจ
- คำร้องเรียนที่รับและแก้ไข
- บริการออนไลน์/ดิจิทัล

### บทที่ 5: การพัฒนาบุคลากร
- การฝึกอบรม จำนวนชั่วโมง
- การศึกษาดูงาน
- รางวัลที่ได้รับ

### บทที่ 6: แผนงานปีถัดไป
- โครงการสำคัญที่จะดำเนินการ
- งบประมาณที่ขอตั้ง
- เป้าหมายที่คาดหวัง

## หลักการเขียนให้น่าอ่าน

### โครงสร้างย่อหน้าผลสำเร็จ
```
[ระบุสิ่งที่ทำ] → [ตัวเลขผลลัพธ์] → [ผลกระทบต่อประชาชน]

ตัวอย่าง:
"เทศบาลดำเนินโครงการซ่อมบำรุงถนนในเขตเทศบาล 12 สาย
ระยะทางรวม 8.4 กิโลเมตร งบประมาณ 3.2 ล้านบาท
ส่งผลให้ประชาชนใน 8 ชุมชนได้รับความสะดวกในการสัญจร
และลดอุบัติเหตุในพื้นที่ร้อยละ 15"
```
